# 🌍 Air Quality Prediction (Jalandhar) using Deep Learning Models

This project focuses on forecasting air quality (PM2.5 levels) for the city of Jalandhar using time-series deep learning models. It is part of a broader multi-city air quality prediction project, where this implementation specifically covers data processing, modeling, and evaluation for Jalandhar.

---

## 📌 Overview

Air pollution is a critical environmental issue impacting public health. This project uses historical air quality data to predict PM2.5 levels for Jalandhar and analyze pollution trends using machine learning and deep learning techniques.

---

## ⚙️ Features

- 📊 Air quality prediction for Jalandhar using time-series data  
- 🤖 Deep learning models: LSTM, GRU, CNN-LSTM  
- 📈 Baseline model: LightGBM  
- 🔍 Model comparison using RMSE, MAE, and R² score  
- 📉 Visualization of predictions and trends  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- TensorFlow / Keras  
- Matplotlib  

---

## 🔄 Workflow

1. Data Preprocessing (cleaning, handling missing values)  
2. Feature Engineering (lag features, time-based features)  
3. Model Training (LSTM, GRU, CNN-LSTM, LightGBM)  
4. Model Evaluation and Comparison  
5. Visualization of results  

---

## 📊 Results

- Achieved strong predictive performance (R² ≈ 0.90)  
- CNN-LSTM showed the best performance  
- Deep learning models effectively captured temporal patterns
